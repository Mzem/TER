# Projet Primalité

Code source et application **(Linux)** dans le répertoire **"App"**.
**Pré-requis :** installer la bibliothèque **GMP**. 
**Exécution :** utiliser un terminal Linux et se positionner sur le répertoire **"App"** :  

* Compilation

		# make

* Exécution

		# make run
	
* Nettoyage binaires

		# make clean
		
* Nettoyage intégral
		
		# make mrproper
